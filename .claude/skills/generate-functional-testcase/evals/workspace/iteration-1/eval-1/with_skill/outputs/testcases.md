## 功能测试用例

### TC-LOGIN-001：[正向] 验证合法手机号和密码登录成功

- **优先级**：P0
- **前置条件**：
  1. 系统正常运行
  2. 已注册手机号 13800138000，密码 abc123456
  3. 账号未被锁定
- **测试步骤**：
  1. 打开登录页面
  2. 输入手机号 13800138000
  3. 输入密码 abc123456
  4. 点击登录按钮
- **预期结果**：
  1. 登录页面正常显示
  2. 手机号输入框显示 13800138000
  3. 密码输入框以掩码形式显示
  4. 跳转至首页，页面顶部显示该用户的昵称
